# SimpleEquipmentManager

Lightweight equipment set manager for World of Warcraft (Classic Anniversary interface 20504). Adds equipment sets on the Character frame.

## Install

1. Copy this folder into your WoW `Interface/AddOns` directory.
2. Enable **SimpleEquipmentManager** on the character select AddOns screen.
3. Open the Character window to manage sets.

## Files

- `SimpleEquipmentManager.toc` — addon metadata and load order
- `SimpleEquipmentManager.lua` — main logic

Settings are stored in the saved variable `SimpleEquipmentManagerDB` (per character/account, in your WoW `WTF` folder, not in this repo).

## Requirements

- Interface: **20504** (see `.toc`)
